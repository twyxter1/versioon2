GIT

'git init' - git repositooriumi tuvastamine projektikausta seest

'git status' - failide oleku tuvastamine

'git add .' - lisa kõik failid jälgimisse (ainult üks fail -> 'git add faili nimi')

'git commit -m "..." - salvesta failiversioon (punktide asemele läheb muudatuse kirjeldus)

'git branch' - vaata olemasolevaid branche

'git log'- vaata muudatuste nimekirja

'git master' - master branchi uuesti sisestamine

'git reset -- hard (muudatuse ID mida tahame kasutada)' - mine selle muudatuse juurde ja kustuta kõik järgnevad

'git checkout -b (branchi nimi)' - Tee uus branch millel on kõik eelnevate branchide commitid

'git checkout master' = 'git master' - muuda master branchiks

'git merge (branchi nimi mida ühendada)' - ühenda branchid

'git branch -D (branchi nimi mille tahad kustutada)' - kustuta branch

NOTES

’git remote add origin (Link)’ – Olemasoleva repositooriumi lisamine

’git remote’ – Laseb teha, vaadata või kustutada repositooriumi ühendusi

’git push -u origin master’ – Lükkab muudatused serverisse

’git branch’ – uue branchi tegemine

’git branch -m’ – branchi nime muutmine

’git clone’ – tee koopia oma repositooriumist

’git fetch’ – repositooriumi allalaadimine kohalikku reposse

’git pull’ – laeb alla kogu repositooriumi ja koheselt uuendab kohalikku repositooriumit
